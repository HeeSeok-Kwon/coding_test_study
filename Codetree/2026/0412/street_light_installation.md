#### 1. 100 100 3 3 10 90
```
최초 가로등 위치 (3, 10, 90)
```

#### 2. 400, 최초 가로등 위치 (3, 10, 90)에서
마을 전체 [1, N] 밝히기 위한 최소 전력
```
|1 - 3| = 2
|3 - 10| = 7 -> 7 / 2 = 3.5
|10 - 90| = 80 -> 80 / 2 = 40
|90 - 100| = 10

max(2, 3.5, 40, 10) = 40 

40 * 2 = 80
```

#### 3. 200, 가로등 추가 (인접한 가로등 사이에 추가)
```
|3 - 10| = 7
|10 - 90| = 80

max(7, 80) = 80

(10 + 90) / 2 = 50

-> 3, 10, 50, 90
```

#### 4. 200, 가로등 추가
```
|3 - 10| = 7
|10 - 50| = 40
|50 - 90| = 40

max(7, 40, 40) = 40

(10 + 50) / 2 = 30
 
-> 3, 10, 30, 50, 90
```

#### 5. 400가로등 위치 (3, 10, 30, 50, 90)에서 마을 전체 [1, N] 밝히기 위한 최소 전력
```
|1 - 3| = 2
|3 - 10| = 7 -> 7 / 2 = 3.5
|10 - 30| = 20 -> 20 / 2 = 10
|30 - 50| = 20 -> 20 / 2 = 10
|50 - 90| = 40 -> 40 / 2 = 20
|90 - 100| = 10

max(2, 3.5, 10, 10, 20, 10) = 20 

20 * 2 = 40
```

#### 6. 300 3, 가로등 제거
```
3, 10, 30, 50, 90
-> 3, 10, 30, 50

Q. 왜 90이 사라졌냐?
각 가로등 위치와 고유번호는 별개
초기에 3, 10, 90이 주어졌고 각 가로등의 고유번호는 1, 2, 3
```

#### 7. 400, 가로등 위치 (3, 10, 30, 50)에서 마을 전체 [1, N] 밝히기 위한 최소 전력
```
|1 - 3| = 2 
|3 - 10| = 7 -> 7 / 2 = 3.5
|10 - 30| = 20 -> 20 / 2 = 10
|30 - 50| = 20 -> 20 / 2 = 10
|50 - 100| = 50

max(2, 3.5, 10, 10, 50) = 50

양 끝(1, 100)에는 가로등이 없으니 2로 나눌 필요가 없음

50 * 2 = 100
```
